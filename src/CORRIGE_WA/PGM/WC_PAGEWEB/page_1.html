Adélia pour le Web

Adelia/Web permet de concevoir et de développer des applications multi-niveaux INTRANET, EXTRANET et INTERNET

Articulé autour des clients java et des architectures JSP, les composants proposés par Adelia/Web apportent une solution globale à la problématique de mise en oeuvre de ces architectures hétérogénes et communicantes.

Adelia/web

  • Génére les clients légers JAVA ou HTML.
  • Assure la gestion de la communication entre les clients et les serveurs Web.
  • Génére les composants java beans utilisés dans les pages JSP.
